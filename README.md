# 🌸 Lily Fashion

A modern and responsive fashion e-commerce web application built with **Next.js**, designed to provide a smooth shopping experience. Users can browse products, search, filter, view product details, and manage their shopping cart.

[🌐 Live Demo:](https://lily-fashion.vercel.app)

## ✨ Features

- 🛍️ Browse fashion products
- 🔍 Search products by name
- 🏷️ Filter products by category
- 💰 Filter products by price range
- 📱 Fully responsive design
- 🎠 Product sliders using Swiper
- ⭐ Product ratings
- 🛒 Shopping cart with Context API
- 💾 Cart persistence using Local Storage
- 📄 Dynamic product details page
- 🎨 Modern UI with Tailwind CSS & DaisyUI

## 🛠️ Tech Stack
### Frontend

- Next.js 16
- Tailwind CSS
- DaisyUI
- Swiper.js
- React Simple Star Rating
- React Icons

### State Management

- React Context API
- Local Storage

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/lily-fashion.git
```

### Navigate to the project

```bash
cd lily-fashion
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
